GSTIN Resolution Logic
1. Look up PPOB for vendor in resolved state
→ If found & Active: use PPOB GSTIN
2. If no active PPOB: look up APOB
→ If found & Active: use APOB GSTIN
3. If neither found:
→ Block PO creation with error
ℹ The resolved GSTIN is read-only. To update, edit the APOB record in Vendor Management.
